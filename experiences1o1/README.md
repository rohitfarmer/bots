# Experiences1o1

Bots for automating social networks related to [https://experiences1o1.carrd.co/](https://experiences1o1.carrd.co/).

**retweet.py**: At every run fetch 250 recent tweets from the home timeline and retweet the ones that are already retweeted by others more than a certain number of times. Skip if the tweet has already been retweeted by the script in the previous run.

**follow.py**: Follow the users that have followed experiences1o1 and contain certain keyword(s) in their bio.

**unfollow.py**: Unfollow the users that are not following experiences1o1 anymore.

**tweet_motivational_quotes.py**: Tweet a motivational quote from a file once a day.
